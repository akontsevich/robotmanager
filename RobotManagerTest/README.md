RobotManagerTest
===========

Set-up:
    * Download Google Test: https://github.com/google/googletest/archive/release-1.8.0.tar.gz
    * Extract it
    * Build and install it

Usage:
    * `mkdir build && cd build`
    * `cmake -DCMAKE_PREFIX_PATH=/path/to/googletest/install ../`
    * `./robotmanagertest`
